Hello! This site contains an assortment of tools to help build and edit creatures for Dungeons and Dragons 5th edition.

If you're looking for the monster scaling tool that used to be at this page you'll find the link below, we've just done a bit of reorganizing now that there are different tools here.

  • Monster Scaler - For scaling monsters up or down to different CRs.
  • Sidekick Builder - For choosing a base creature and building a sidekick.

Creature statistics from the SRD are property of Wizards of the Coast and are used here in accordance with the OPEN GAME LICENSE.
