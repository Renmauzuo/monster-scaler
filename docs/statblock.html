Using this tool

This tool allows you to enter a JSON object of any creature/character and render it as a statblock. You can use this file for an example of a basic statblock.

You can also use the JSON exports from other tools as a starting point. Not that these will contain some extra fields (such as attack strings) which this tool does not need, as it derives them itself. However, while unnecessary they won't cause any harm if you leave them in the statblock.

Creature statistics from the SRD are property of Wizards of the Coast and are used here in accordance with the OPEN GAME LICENSE.
