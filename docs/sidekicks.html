Using this tool

This tool was created to assist in building sidekicks based on the sidekick rules in Unearthed Arcana. To start select a creature to use as a base. The chosen creature will automatically be scaled to CR 1/2, the maximum allowed per RAW. Sidekick stats will then be added on top of the monster stat block.

Creature statistics from the SRD are property of Wizards of the Coast and are used here in accordance with the OPEN GAME LICENSE.
